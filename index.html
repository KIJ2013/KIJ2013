KIJ2013 Smartphone App Support

KIJ2013

KIJ2013 is an International Scout Jamboree to be held in Kent in August 2013. It will be attended be over 4000 participants representing more than a dozen countries.

Smartphone App

As part of the Jamboree a smartphone app has been produced to complement the experience. The source code for the app has been published on github.

Technical Support

These pages are intended to give an overview to both the design and development decisions made. In the latter capacity these pages should serve as a support resource for the app. Material will be mainly from a technical perspective. To some extent this documentation will constitute a specification.

Source Code

The source code can be obtained with:
